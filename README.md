პროექტი "ინფო კალენდარი"
ეს არის საგანმანათლებლო ჟურნალი, რომელიც გთავაზობთ ყოველდღიურ ინფორმაციას ათასობით მსოფლიო დღესასწაულების
(საქართველოს, მსოფლიო და მართლმადიდებლური ეკლესიის), ქალაქების დღეების, მნიშვნელოვანი ისტორიული მოვლენების,
ცნობილი ადამიანებისა და ისტორიული პიროვნებების დაბადება/გარდაცვალების თარიღებისა და სხვა მოვლენების შესახებ,
რომლებიც დღევანდელ დღეს უკავშირდება.
